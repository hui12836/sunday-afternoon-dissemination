## 裝飾性  KPI
KPI 是「 Key Performance Indicator 」，中文可以理解成「關鍵指標」；原本它常用在儀表板或後台，用來顯示真正重要的數據，例如轉換率、流量、完成率、參與率等。 [2][8][9]

放到你的網站裡，假設你有一區用大字顯示幾個數字，例如幾個服務量、倡議次數或某種成果，但沒有明確告訴人「這數字代表什麼、從哪裡來、為什麼重要」，那它就是我說的裝飾性  KPI 。對介紹臉部平權的首頁來說，這種數字如果不夠精準，反而會讓人分心，不如改成「三大工作重點」或「三件你現在就能理解的事情」更有效。 [7][2]
你可以這樣分辨：   
- 真正的  KPI = 有目的、有定義、有來源。   
- 裝飾性  KPI = 有大數字，但沒有清楚意義。 [2][8]

icon 系統不是單指某一顆  icon ，而是指「整個網站使用圖示的方法有沒有規則」。一個好的  icon 系統會讓所有圖示在大小、線條粗細、圓角、風格、填色方式上都一致，這樣使用者看起來才會覺得是同一套介面語言，也更容易辨認用途。 [3][10]
例如一套  icon 系統通常會先決定幾件事：   
- 圖示都用線框，還是都用實心。   
- 線條粗細是不是一致。   
- 圖示外框是不是都用圓角。   
- 顏色是不是同一規則，例如平常白色、滑過變黃色。   
- 尺寸是不是固定，例如都用  24px 或  20px 。 [3][10]
放到你的網站裡，我前面提「要改成一致的  icon 系統」，意思是你不要一個地方用圓形彩色  icon 、一個地方用文字縮寫  FB/IG 、一個地方又像表情符號，這會讓視覺語言不統一。你可以選一套風格，例如全部改成簡潔線框  icon ，再統一尺寸、顏色與  hover 狀態，整體專業感會差很多。 [7][3][10]
