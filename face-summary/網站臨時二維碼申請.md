## 為網站申請臨時 QR Code

有幾個免費、快速的方式可以產生臨時 QR Code：

---

### 🔧 免費線上工具（最快速）

| 工具 | 特色 |
|------|------|
| [qr-code-generator.com](https://www.qr-code-generator.com) | 免費、可下載 PNG/SVG |
| [goqr.me](https://goqr.me) | 簡單易用，支援多種格式 |
| [qrcode-monkey.com](https://www.qrcode-monkey.com) | 可自訂顏色與 Logo |

使用步驟：
1. 進入網站
2. 貼上你的網址（URL）
3. 點擊「生成」
4. 下載或截圖儲存

---

### 💻 用 Google 快速產生

在 Chrome 瀏覽器開啟你的網站後：
- 點擊網址列右側的「**分享**」圖示
- 選擇「**建立 QR 圖碼**」
- 直接下載即可

---

### 🐍 用程式碼自動產生（Python）

```python
pip install qrcode[pil]
```

```python
import qrcode
img = qrcode.make("https://你的網址.com")
img.save("qrcode.png")
```

---

### 注意事項

- **臨時用途**：直接用線上工具產生靜態 QR Code 即可，不需要帳號
- **需要追蹤掃描次數**：可用 [Bitly](https://bitly.com) 先縮短網址再產生 QR Code
- **長期使用**：建議用動態 QR Code 服務（如 QR Tiger、Beaconstac），方便日後修改連結
